# SITL launcher

## arguments

```
ros2 launch -s  sitl_launcher demo.launch.py
Arguments (pass arguments as '<name>:=<value>'):

    'use_rviz':
        Whether to start RVIZ
        (default: 'False')

    'use_qgroundcontrol':
        Whether to start QGroundcontrol
        (default: 'False')

    'sitl_world':
        World [yosemite, mcmillan, ksql, baylands]
        (default: 'yosemite')
```
